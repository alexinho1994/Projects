Am creat o aplicatie QuizApp cu intrebari din fotbal.
Am folosit o interactiune client-server pentru logare si inregistrare.
Inaintea rularii trebuie modificata adresa IP din clasa NetworkManager, in constructor.
Am introdus intr-o baza de date intrebarile, variantele de raspuns si raspunsurile corecte.
In layout-ul activitatii apare intrebarea, 3 butoane pentru variante de raspuns, un buton
wikipedia si un buton next. Daca apasam pe Wikipedia, se deschide browserul cu pagina wikipedia
pentru Uefa Champions League 2016-2017. Butonul pentru next functioneaza doar daca a fost apasat 
raspunsul corect pentru intrebare si trimite urmatoarea intrebare.

Raspunsurile sunt a-b-c-c-a-a
